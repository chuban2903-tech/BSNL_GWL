🔗 Connect Google Sheets

The app fetches live data directly from Google Sheets every time you open a site. You must deploy the Apps Script first.

  1. Open your Google Sheet → Extensions → Apps Script
  2. Delete ALL old code → paste the new APPS_SCRIPT_CODE.js → Save
  3. Deploy → New Deployment → Web App
  4. Execute as: Me  |  Access: Anyone
  5. Authorise → copy the /exec URL
